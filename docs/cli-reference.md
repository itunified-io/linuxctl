## Conventions

- Every subsystem manager command exposes `plan`, `apply`, `verify` verbs.
- `linuxctl apply ...` orchestrates the full 13-manager DAG.
- Persistent flags `--env`, `--host`, `--yes`, `--dry-run`, `--format` apply across the tree.
